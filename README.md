# FitPulse

A personal project for food classification and health tracking, focused on improving individual lifestyle efficiency through AI-driven insights.

## Core Features
* **AI-Driven Classification:** Intelligent categorization of food items.
* **Health Metrics Tracking:** Seamless monitoring of daily nutritional intake.
* **Tech Stack:** WeChat Mini-Program, React Native (Expo), and Cloud AI API integration.

## Development
This project is part of a "Digital Symbiote" experiment, utilizing long-chain reasoning to automate complex UI layouts and engineering tasks.
